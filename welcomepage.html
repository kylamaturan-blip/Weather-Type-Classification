📘 System Overview

This system collects weather-related inputs such as temperature, humidity, wind speed, pressure, and visibility, then processes them using a trained machine learning model to classify the current weather condition.

🌦 Weather Input Module

Accepts user-provided atmospheric data such as temperature, humidity, wind speed, pressure, and visibility. These inputs are validated and prepared before being sent to the classification model.

Ensures that all data entries are within acceptable ranges to maintain prediction accuracy and system reliability.

🤖 Classification Model

Uses a trained machine learning model to classify weather conditions into four categories: Sunny, Rainy, Snowy, and Foggy.

The model analyzes patterns from the dataset and generates predictions based on the relationship between atmospheric variables.

💾 Data Storage

All prediction results and input data are stored in a Firebase database for future reference and analysis.

This allows users to track previous predictions, review logs, and monitor system performance over time.

📊 Dataset Integration

The system is trained using a weather classification dataset sourced from Kaggle, containing multiple atmospheric features and labeled outputs.

This dataset helps the model learn patterns and improve its ability to classify weather conditions accurately.
